== Description ==

A Collection of Pure CSS and JS Creations!

= About CSSDeck =

[CSSDeck](http://cssdeck.com) is a big collection of items built with pure CSS and HTML to help frontend coders realize the power and flexibility of CSS. Hopefully the creations will help them get better at the new features of CSS (and in general too) and they'll start making more use of it in their projects.

= CSSDeck WordPress Plugin =

Using this WordPress plugin you can easily embed any item from CSSDeck into your WordPress site using the item URL.

Choose any item from CSSDeck, or create your own! Copy the URL and paste it to the post text editor. Then simply click over to the visual editor to confirm that it loads properly.

== Installation ==

= Installation =
1. In your WordPress Dashboard go to "Plugins" -> "Add Plugin".
2. Search for "CSSDeck".
3. Install the plugin by pressing the "Install" button.
4. Activate the plugin by pressing the "Activate" button.

= Minimum Requirements =
* WordPress version 3.5 or greater.
* PHP version 5.2.4 or greater.
* MySQL version 5.0 or greater.

= Recommended  Requirements =
* The latest WordPress version.
* PHP version 7.0 or greater.
* MySQL version 5.6 or greater.
